1. Сначала поднимаем докер контейнер через команду " docker-compose up " (выполнять находясь в проекте с .yml файлом)
2. Нужно установить dotnet-ef Для создания миграций команда    dotnet tool install --global dotnet-ef
3. Вводим команду dotnet ef в папке проекта
4. Иницилизаровать миграции для этого вводим команду dotnet ef migrations add ИМЯ_МИГРАЦИИ -s НАЗВАНИЕ_ПРОЕКТА -p НАЗВАНИЕ_ПРОЕКТА_ГДЕ_ЛЕЖИТ_ДБ_КОНТЕКСТ
5. Применяем миграции командой dotnet ef database update -s НАЗВАНИЕ_ПРОЕКТА -p НАЗВАНИЕ_ПРОЕКТА_ГДЕ_ЛЕЖИТ_ДБ_КОНТЕКСТ
